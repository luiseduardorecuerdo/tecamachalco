ETIQUETAS HTML

etiqueta funcion ejemplo
span abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. (span)(/span)
br produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante. (br)(/br)
a Es una etiqueta que nos ayuda a poder crear un enlace a una página web. (a)(/a)
img se usa para colocar una imagen a la pagina (/img)
b Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando. Sus etiquetas son: y (b)(/b)
del marca las partes de un texto o documento que han sido suprimidas o sustituidas (del)(/del)
ins marca las partes de un texto que han sido añadidos al documento (ins)(/ins)>
sub efine un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. (sub)(/sub)
sup efine un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice (sup)(/sup)
strong El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto (strong)(/strong)
i es un espacio de texto que representa un cambio en el estado de ánimo o la calidad del texto. Este texto se muestra en cursiva. (i)(/i)
em es el apropiado para marcar con énfasis las partes importantes de un text (em)(/em)
mark presenta un texto marcado o resaltado como referencia o anotación, debido a su relevancia (mark)(/mark)
small ace el tamaño del texto una talla más pequeña (small((/small)
